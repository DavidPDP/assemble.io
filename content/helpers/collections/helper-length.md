## \{{length}}

> Returns the length of the collection.

Parameters: `none`

Data:

```json
"collection": [
  "Amy Wong",
  "Bender",
  "Dr. Zoidberg",
  "Fry",
  "Hermes Conrad",
  "Leela",
  "Professor Farnsworth",
  "Scruffy"
]
```

Template:

```handlebars
\{{length collection}}
```

Renders to:

```handlebars
8
```
